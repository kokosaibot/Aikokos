# Avalava-style Telegram Mini App Base

Файлы для GitHub Pages / Telegram Mini App.

## Как использовать
1. Залей , ,  в папку  твоего репозитория.
2. Сделай commit + push.
3. Дождись обновления GitHub Pages.
4. Проверь ссылку в браузере и в Telegram-боте.

## Что внутри
- Главная как на референсе
- Картинка
- Видео
- Modal выбора модели
- Modal выбора ratio
- Modal тарифов
- Профиль
- Инструменты
